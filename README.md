# MOMIJI Terminal 🍁

**AI ネイティブなターミナルアプリ**（macOS）。Claude Code・Codex・Gemini などの AI CLI エージェントを、分割ペイン・タブ・テーマ・記憶ログなどと一緒に快適に使えます。

- 📘 使い方ガイド: **https://momiji-docs.web.app**
- 💴 料金・プラン: **https://momiji-docs.web.app/pricing**

---

## ⬇️ ダウンロード（最新版）

右側の **[Releases](../../releases/latest)** から、最新の `MOMIJI_x.x.x_aarch64.dmg` をダウンロードしてください。

## 動作環境

- **macOS 12 (Monterey) 以降**
- **Apple Silicon（M1 以降）専用** ※Intel Mac には対応していません

## インストール手順

1. `MOMIJI_x.x.x_aarch64.dmg` をダウンロードして開く
2. 表示された **MOMIJI.app** を **アプリケーション** フォルダにドラッグ＆ドロップ
3. 初回起動時、macOS のセキュリティ警告が出る場合があります（下記参照）

### ⚠️ 「開けません」と表示される場合（初回のみ）

本アプリは Apple の公証（Notarization）を受けていないため、初回起動時に Gatekeeper の警告が出ます。以下のいずれかで開けます。

**方法A（かんたん・推奨）**
1. アプリケーションフォルダの **MOMIJI.app を右クリック → 「開く」**
2. 警告ダイアログの **「開く」** をクリック（2回目以降は通常どおり起動できます）

**方法B（方法Aで開けない場合）**
「ターミナル」アプリで次のコマンドを実行してから起動してください：

```bash
xattr -dr com.apple.quarantine /Applications/MOMIJI.app
```

> macOS の設定によっては **システム設定 → プライバシーとセキュリティ** の下部に「このまま開く」ボタンが出ることもあります。

## はじめての方へ

- アプリ内の **設定 → アカウント** から Google ログインすると、プランやAI機能が利用できます。
- 無料プランでもターミナルの基本機能（複数 AI エージェント・分割・タブ・テーマ・Lark CLI 連携など）はフルに使えます。
- 各プランで何ができるかは **[料金・プラン](https://momiji-docs.web.app/pricing)** をご覧ください。
